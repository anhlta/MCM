Forgot password:
- User clicks on Forgot password link to request for password reset
- User is asked to provide his account name (username)
- Error message will appear if input username doesn't exist
- After clicking on the button, user will receive an email with a unique link which allows him to change his password 
- The link in email is valid for only 24 hours. The page will show error message if user clicks on an expired link.
- User clicks on the link to go to change password page. 
- To change password, user is required to reenter his username for validation, new password and confirm new password
- If all infomration is valid, user password will be changed
